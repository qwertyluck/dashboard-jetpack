# 🚀 Dashboard Jetpack - Pusat Generator Dokumen Resmi

Dashboard terpusat berbasis web yang dirancang untuk mengintegrasikan berbagai aplikasi generator dokumen resmi instansi dalam satu pintu.

## 📋 Fitur Utama
* **Dashboard Terpadu:** Antarmuka modern dengan navigasi samping yang bersih dan responsif.
* **Generator SPPD:** Modul khusus untuk pengisian dan pembuatan Surat Perintah Perjalanan Dinas.
* **Generator SPTJM:** Modul untuk menghasilkan Surat Pernyataan Tanggung Jawab Mutlak.
* **Generator Nota Dinas:** Modul untuk menyusun format Nota Dinas resmi secara cepat.

## 🛠️ Teknologi yang Digunakan
* **HTML5 & JavaScript** (Frontend murni tanpa instalasi rumit)
* **Tailwind CSS** (Kerangka desain antarmuka instan dan modern)
* **Docxtemplater & PizZip** (Pustaka pemrosesan dokumen Word dinamis)
